About the projectl

A live chat that allows users to communicate and exchange messages quickly and securely was created using PHP as a development basis. The site is distinguished by the ease of updating users' personal photos, and the immediate ability to send messages quickly and in an encrypted manner to ensure privacy and security.

project features:

  • One of the notable features of the project is that there is no need for a database, as the data is stored encrypted in a text file. The project is also open source, allowing developers to modify, improve and adapt it to their needs.
  • Open source project

Technologies used:

PHP: PHP was used as a programming language to develop the algorithms and back-end logic of the website
jQuery: jQuery is used to simplify interactions and event management in the user interface
AJAX: AJAX is used to load data without reloading the page, making the user experience smoother and faster.
